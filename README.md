# Description

The Project is inspired by the sorting visualizer made by Clement Mihailescu on YouTube.
The above project includes algorithms as follows:

 - Quick Sort
 - Merge Sort
 - Heap Sort
 - Insertion Sort
 - Bubble Sort
 - Shell Sort 
 - Cocktail Sort
 - Comb Sort
 - Odd-Even Sort
 - Radix Sort
 - Bucket Sort
 - Counting Sort
